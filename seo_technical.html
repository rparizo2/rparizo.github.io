SEO Type: Technical

Technical SEO ensures that a website meets the technical requirements of modern search engines, such as a fast load time, mobile-friendliness, and secure connections.

Key Terms in Technical SEO

Crawlability: The ability of search engines to discover pages on a website. Factors affecting crawlability include internal linking, URL structure, and the robots.txt file.

Indexability: The ability of search engines to store and rank pages. This involves meta tags, canonical tags, and sitemaps.

Core Web Vitals (CWV): Metrics that measure user experience related to speed, including First Input Delay (FID), Largest Contentful Paint (LCP), and Cumulative Layout Shift (CLS).

Canonical URL: The preferred version of a URL, used to resolve duplicate content issues.

HTTPS: A secure version of HTTP, which is a ranking factor for Google and ensures data security.

AMP (Accelerated Mobile Pages): A framework to create faster mobile pages, often resulting in a separate AMP version of a website.

Schema Markup: Code added to a website to help search engines understand the content and provide rich results in SERPs.

XML Sitemap: A file that lists all the pages of a website, helping search engines crawl and index the site more effectively.

Robots.txt: A file that tells search engines which pages or sections of a website should not be crawled.

Page Speed: The time it takes for a web page to load, which affects user experience and search engine rankings.
